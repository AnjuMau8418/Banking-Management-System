# Project Overview

This Banking Management System simulates a basic banking environment using C++.

The system allows users to:

- Create bank accounts
- Login securely
- Perform transactions
- Store data persistently using text files

## Architecture

The system follows modular OOP design:

- Bank.cpp → Controls core system logic
- Account.cpp → Handles account data structure
- Transaction.cpp → Logs transactions
- File System → Stores persistent data

## Design Concepts Used

- Object-Oriented Programming
- Encapsulation
- File Handling
- Data Persistence
- Menu-driven CLI Application
